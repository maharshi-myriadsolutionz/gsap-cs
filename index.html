Basic Animations

Core GSAP methods: to(), from(), fromTo()

gsap.to()
gsap.from()
gsap.fromTo()
// gsap.to()
gsap.to("#basic-box", {
    x: 200,
    rotation: 360,
    duration: 2,
    ease: "power2.out"
});

// gsap.from()
gsap.from("#basic-box2", {
    y: -100,
    opacity: 0,
    duration: 1.5
});

// gsap.fromTo()
gsap.fromTo("#basic-box3", 
    { scale: 0, rotation: -180 },
    { scale: 1, rotation: 0, duration: 1 }
);

Timeline

Sequence animations with gsap.timeline()

2
3
const tl = gsap.timeline();

tl.to("#tl-box1", { x: 100, duration: 1 })
  .to("#tl-box2", { y: 100, duration: 1 }, "-=0.5")
  .to("#tl-box3", { rotation: 360, duration: 1 }, "+=0.2");
  
// Control timeline
tl.play();
tl.pause();
tl.reverse();
tl.restart();

Staggering

Animate multiple elements with stagger

gsap.from(".stagger-item", {
    scale: 0,
    opacity: 0,
    duration: 0.5,
    stagger: {
        amount: 1.5,
        from: "center",
        grid: [3, 3]
    },
    ease: "back.out(1.7)"
});

ScrollTrigger

Trigger animations on scroll

Scroll to see me!
I'm pinned!
Scrubbed animation 
// Basic scroll trigger
gsap.to("#st-box1", {
    x: 200,
    scrollTrigger: {
        trigger: "#st-box1",
        start: "top 80%",
        end: "bottom 20%",
        scrub: true,
        markers: true
    }
});

// Pinned section
ScrollTrigger.create({
    trigger: "#st-box2",
    pin: true,
    start: "top center",
    end: "+=300"
});

DrawSVG

Animate SVG stroke paths

// Draw SVG path
gsap.fromTo("#svg-path", {
    drawSVG: "0%"
}, {
    drawSVG: "100%",
    duration: 2,
    ease: "power2.inOut"
});

// Draw circle
gsap.fromTo("#svg-circle", {
    drawSVG: "0%"
}, {
    drawSVG: "100%",
    duration: 1.5,
    delay: 0.5
});

Draggable

Make elements draggable with bounds and inertia

Drag me!
Draggable.create("#drag-box", {
    type: "x,y",
    bounds: ".drag-bounds",
    inertia: true,
    onDrag: function() {
        console.log("dragging");
    },
    onThrowUpdate: function() {
        console.log("throwing");
    }
});

// Typewriter effect
gsap.to("#typewriter", {
    text: "Welcome to GSAP!",
    duration: 2,
    ease: "none"
});

gsap.to("#counter", {
    textContent: 100,
    duration: 2,
    snap: { textContent: 1 },
    ease: "power2.out"
});

GSDevTools

Visual timeline scrubber for debugging

Debug me!
const tl = gsap.timeline();
tl.to("#devtools-box", { x: 200, duration: 1 })
  .to("#devtools-box", { y: 100, duration: 1 })
  .to("#devtools-box", { rotation: 360, duration: 1 });

// Create dev tools
GSDevTools.create({
    animation: tl,
    globalSync: false
});
